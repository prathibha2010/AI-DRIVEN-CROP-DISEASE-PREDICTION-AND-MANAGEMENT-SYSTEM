# AI-DRIVEN-CROP-DISEASE-PREDICTION-AND-MANAGEMENT-SYSTEM
AI-driven Crop Disease Prediction &amp; Management system that identifies plant diseases from leaf images. The system provides disease names, severity, and recommended treatments or fertilizers. Helps farmers make informed decisions, reduce crop loss, and improve yield with accurate, real-time insights.
This study introduces an AI-powered decision support system designed to improve key agricultural 
tasks such as fertilizer recommendation, crop selection, and crop disease detection. It uses machine 
learning—specifically the Random Forest algorithm—for predicting suitable crops and fertilizers 
based on soil nutrients (N, P, K), pH, temperature, rainfall, humidity, and soil type. Data preprocessing 
techniques like normalization, label encoding, and handling missing values were used to enhance 
model accuracy. For disease detection, a Convolutional Neural Network (CNN) was trained on over 
50,000 images of healthy and diseased leaves, achieving high classification performance. The system 
is accessible through a simple web interface, providing real-time, data-driven insights to farmers. 
Overall, this integrated platform supports sustainable, efficient, and modern precision agriculture by 
helping farmers make better decisions and reduce resource wastage
